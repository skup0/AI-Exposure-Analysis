DATA AVAILABILITY

The raw datasets used in this analysis are publicly available from Statistics Canada
but are not included in this repository due to file size constraints.

To reproduce the analysis, download the following datasets and place them in the
directory:

    raw-data/

Required files:
- 2012-wages.csv
- 2018-wages.csv
- 2024-wages.csv
- skill-gap.csv
- gender.csv

Data sources:
- Occupational Wage Data (2012, 2018, 2024):
  Statistics Canada — Labour Force Survey / Job Vacancy and Wage Survey

- Skill Gap Data:
  Statistics Canada — Actions planned by businesses or organizations to address
  skills gaps or employee skill deficiencies

- Gender Workforce Data:
  Statistics Canada — Employment by gender and occupation

After downloading the datasets and placing them in `raw-data/`, run the notebook
from top to bottom to reproduce all results and figures.
